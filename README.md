# Smart-book-AI
​"SmartbookAI: نظام ذكاء اصطناعي لتنسيق النصوص (Text Formatting AI)، يوفر تحكماً ديناميكياً في مستوى ظهور النص بناءً على نمط الترقيم:​الترقيم اللاتيني (1.): للنصوص البارزة (High Emphasis).​الترقيم العربي (١.): للنصوص الثانوية (Medium Emphasis).​بدون ترقيم: للنصوص الاعتيادية (Normal Text)."
